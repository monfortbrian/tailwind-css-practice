# Tailwind CSS Brush-Up

Building UI components with Tailwind CSS - from Figma design systems to production-ready code. Preparing for component libraries like shadcn/ui and Origin UI.

## Projects

### 1. Flight Booking Card
Flight itinerary card with trip details, gate info, boarding QR code, seat assignment, and countdown timer.

**[View Figma Design →](https://figma.com/file/your-flight-card-link)**

### 2. Job Listing Card  
Job post card with role details, hourly rate, category tags, user profile, and apply button.

**[View Figma Design →](https://figma.com/file/your-job-card-link)**

## Getting Started
```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Tech Stack

- Tailwind CSS v4.1.18 (Utility-first CSS framework)
- Vanilla HTML/CSS/JS
